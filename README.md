# 39 Journey | Experiential Microsite

A scrollytelling web experience that translates a static "Thank You" poster into a dynamic, emotional digital journey. Built as a tribute to the First Sound of the Future and the brilliant minds (Vocaloid-P) who gave her a soul.

🌐 **[Live Demo: 39 Journey](https://mufuyumoku.github.io/39/)**

---

## 📖 The Concept
In traditional Japanese numeric wordplay (*Goroawase*), the sequence **39** can be read as **'San-Kyu'** (phonetically mirroring the English "Thank You"). 

However, in the digital expanse, **3** and **9** can also be read as **'Mi-Ku'**. This microsite bridges that mathematical logic with human empathy, creating a narrative that shifts from a cold, analog dictionary definition into a warm, synthesized resonance.

## 🛠️ Tech Stack
This project is built purely with frontend technologies, focusing on fluid animations and responsive experiential design without heavy frameworks:
* **HTML5 & CSS3** (Fluid typography, custom CSS variables, noise overlays)
* **Vanilla JavaScript** * **GSAP (GreenSock Animation Platform)** & **ScrollTrigger** (For high-performance, replayable scrollytelling animations)
* **Dynamic SVG Manipulation** (For the drawn chaos effect)

## 📂 Project Structure
* `index.html` - The semantic backbone divided into 6 distinct "Acts".
* `style.css` - Handles the analog-to-digital theme transitions, layouting, and mobile responsiveness.
* `script.js` - The engine powering the GSAP timelines, pinned scroll events, and the dynamic multilingual comment cloud generator.
